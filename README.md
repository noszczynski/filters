This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project targets

- [ ] the product can have any filters
- [ ] a product can belong to one category
- [ ] a category can have another category as its parent
- [ ] filters for a product should be suggested by categories
- [ ] a product can have several values for one filter
- [ ] one filter can be shared between categories
